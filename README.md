随机选人器



利用随机数生成器选人，序号和人名对应，随机数生成器生成随机数字，然后输出对应的人名。



1.根据用户输入的总人数，创建数组，读取名单，将名字写入结构体数组。

2.用户输入指定的抽取人数，创建循环，生成随机数。 

3.根据用户指定的抽取人数循环若干次，每次生成不同的随机数，并输出对应的人名
